# Information Security Risk Assessment Report

## Overview
This project is an independently prepared information security risk assessment for a fictional scenario — FinTech Solutions Ireland Ltd, a mid-sized financial technology company processing payments for 150,000 customers across the EU. I conducted this assessment using the NIST SP 800-30 risk assessment methodology and ISO 31000:2018 risk management principles.

## Scenario
- **Organisation:** FinTech Solutions Ireland Ltd (fictional)
- **Sector:** Financial Technology / Payments Processing
- **Location:** Dublin, Ireland
- **Regulatory Environment:** GDPR, PSD2, DORA (Digital Operational Resilience Act)

## Risk Scoring Methodology
Risks are scored on a 5x5 Likelihood × Impact matrix:
- **CRITICAL:** Score 20–25
- **HIGH:** Score 12–19
- **MEDIUM:** Score 6–11
- **LOW:** Score 1–5

## Report Contents
- Executive summary with risk distribution overview
- Full methodology explanation (NIST SP 800-30 + ISO 31000)
- 5x5 risk matrix (colour coded)
- Detailed risk register with recommended controls and owners
- Consolidated risk summary table
- Prioritised remediation roadmap
- GDPR and DORA regulatory compliance notes
